City Weather est une application Web permettant de connaître la méteo d'une ville donnée en utilisant les informations du site OpenWeather via leur API.

Elle se compose de 3 onglets :

- Page Home : Cette page contient le coeur de l'application et permet de consulter la météo d'une ville. Pour cela, il faut rentrer le nom de la ville voulue dans le champ de texte et appuyer sur le bouton situé en dessous. Les informations essentielles de la météo actuelle apparaissent alors. Il est possible d'obtenir des informations plus poussées en cliquant sur la 3ème ligne (un texte sur la droite vous indique la ligne). Il est aussi possible de consulter la météo prévue pour les prochaines heures (3 heures en avance) en cliquant sur le texte situé en bas des informations actuelles, et de consulter les informations plus poussées de ces prévisions de la même facon.

- Page Contact : Vous pouvez me contacter à l'adresse mail donnée en cas de question à propos de l'application.

- Page About : Court descriptif du contexte de cette application Web.
